--------------------------------------------------------------------------------
SETUP INSTRUCTIONS
--------------------------------------------------------------------------------

1. Open a terminal and navigate to this project folder.

2. (Optional) Create a virtual environment:
   python -m venv .env  --system-site-packages

   For Mac/Linux:
       source .env/bin/activate #(if using bash/zsh)
       source .env/bin/activate.fish #(if using fish)

   For Windows:
       .\env\Scripts\activate

3. Install the required packages using pip:
       pip install -r requirements.txt

4. Run the application from the folder 510414346_Sound_Processing_Assignment/
       python Final_Cochlear_Stimulator_Code/main/main.py
